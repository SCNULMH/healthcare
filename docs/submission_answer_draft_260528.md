검진AI 리셋코치는 건강검진 결과를 받은 사용자가 자신의 만성질환 위험을 이해하고 생활습관 개선을 시작하도록 돕는 웹 기반 예방관리 서비스입니다. 많은 사용자는 건강검진표의 혈압, 공복혈당, 콜레스테롤 수치를 확인해도 어떤 수치가 중요한지, 본인에게 어떤 행동 변화가 필요한지 판단하기 어렵습니다. 또한 허리둘레나 지질 수치처럼 기억하지 못하는 항목이 있으면 기존 서비스에서는 분석 자체가 막히거나 부정확한 기본값으로 채워지는 문제가 있습니다.

본 서비스는 이런 문제를 해결하기 위해 검진 입력, 생활 입력, AI 분석, 결과 설명, 개선 행동, 저장/변화 확인을 하나의 흐름으로 제공합니다. 사용자는 직접 검진 수치를 입력하거나 공공데이터 샘플을 불러올 수 있고, OCR을 통해 건강검진표 이미지/PDF에서 입력 후보를 자동 반영할 수 있습니다. 분석 결과는 고혈압·당뇨·이상지질혈증 위험도로 제공되며, 위험도를 높인 요인과 낮추기 위한 행동이 함께 표시됩니다.

핵심 목표 시장은 건강검진을 받은 성인, 만성질환 전단계 관리가 필요한 사용자, 건강 앱 연동 전에도 웹에서 간편하게 검진 기반 예방관리를 시작하고 싶은 사용자입니다. 서비스는 의료 진단이나 치료 결정을 대체하지 않고, 건강검진 기반 예방관리 참고 정보와 의료진 상담 권고를 제공합니다.

## 4. 사업계획서 3. 독창성·발전가능성·완성도

### 독창성

검진AI 리셋코치의 차별점은 “위험도 예측 → 이유 설명 → 오늘의 행동”을 한 번에 연결한다는 점입니다. 기존 서비스는 수치 기록, 운동 기록, 일반 건강정보 제공에 머무르는 경우가 많습니다. 본 서비스는 공복혈당, 혈압, 지질 수치, BMI, 생활패턴을 통합해 질환별 위험도를 산출하고, 사용자가 바로 실행할 수 있는 작은 행동으로 변환합니다.

또한 결측치 처리 방식이 독창적입니다. 사용자가 수축기혈압만 알고 이완기혈압은 모르는 경우에도 입력된 혈압값은 판단에 반영합니다. 지질 수치를 모두 모르는 경우에는 총콜레스테롤·HDL·LDL·중성지방 직접 판단을 제외하고, 외식·야식·운동·걸음수 등 생활요인을 기반으로 위험을 추정합니다. 이는 사용자가 완전한 검진표를 가지고 있지 않아도 서비스 접근성을 유지합니다.

### 발전가능성

단기적으로는 B2C 예방관리 웹앱으로 운영 가능하며, 중기적으로는 건강검진 수검자 대상 리포트 서비스, 기업 임직원 건강관리, 지자체 건강증진사업 연계로 확장할 수 있습니다. 장기적으로는 의료 마이데이터, 건강정보고속도로, HealthKit/Health Connect와 연동해 검진 기록·걸음수·활동량 자동 입력 기반의 개인화 서비스를 제공할 수 있습니다.

수익 모델은 개인 프리미엄 리포트, 기업·기관 건강관리 SaaS, 보건소·지자체 예방관리 프로그램 연계, 보험·헬스케어 파트너십 등을 검토할 수 있습니다. 단, 민감 건강정보를 다루므로 사용자 동의, 최소수집, 암호화, 비식별 통계 활용 원칙을 우선합니다.

### 완성도

현재 시제품은 FastAPI 기반 백엔드와 정적 웹 UI로 구현되어 있습니다. 주요 기능은 건강검진 수치 입력, 생활패턴 입력, 항목별 “모름” 처리, 위험도 분석, 결과 탭, 평균 비교, 신뢰도 카드, 개인화 추천, Firebase 계정·프로필·이전 진료기록 저장, 진단결과 수동 저장, OCR 입력 보조, Render 배포 설정입니다. 저장 API는 로그인 세션 토큰이 없으면 서버에서 차단하도록 보강했습니다.

검증은 Python 유닛 테스트와 JavaScript 문법 검사를 통해 수행했습니다. 결측치 fallback, 음주 변환, Firebase credential fallback, OCR 모름 자동 연동, 저장 인증 흐름을 테스트 케이스로 포함했습니다.

공공 건강검진 데이터를 단순 조회가 아니라 개인 맞춤 예방관리 UX로 전환합니다. 결측치 fallback, OCR 입력 보조, 설명 가능한 AI, 기록 기반 변화 비교를 결합해 실제 사용자 환경에서도 지속 가능한 건강관리 시제품을 제시합니다.

### 산업적 파급효과

검진기관, 기업 건강관리, 지자체 건강증진사업, 보험·헬스케어 플랫폼과 연계할 수 있는 확장성이 있습니다. 수검자는 검진 결과를 이해하고 행동으로 연결할 수 있고, 기관은 예방관리 프로그램 참여율과 사후관리 효율을 높일 수 있습니다.

### ESG 혁신

건강정보 접근성과 이해도를 높여 예방 중심의 사회적 가치를 창출합니다. 웹 기반 서비스로 별도 앱 설치 없이 접근 가능하며, 종이 안내문 중심의 건강관리 정보를 디지털 리포트로 전환할 수 있습니다. 또한 과도한 의료 진단 표현을 피하고 사용자 동의·최소수집·비저장 OCR 원칙을 적용해 윤리적 AI 활용을 지향합니다.

## 8. 시제품 증빙 자료 구성안

필수 캡처:

1. 홈 화면: 최근 건강 리셋 점수와 이전 기록 영역
2. 계정 화면: 로그인/회원가입 및 로그인 후 폼 숨김 상태
3. 기본정보 화면: 나이, 성별, 키, 체중, 허리둘레 입력
4. 검진 화면: 혈압·혈당·지질 수치와 항목별 모름 체크
5. 생활 화면: 아침식사, 음주 주간/월간/잔 수, 야식, 외식, 흡연
6. OCR 화면: 파일 업로드 및 원본 비저장 안내
7. 결과 기준 탭: AI 판단 단계, 생활요인 점수 기준, 평균 비교
8. 위험도 탭: 고혈압·당뇨·이상지질혈증 위험도 카드
9. 신뢰도 탭: 입력값 완성도, 학습 평균 비교
10. 추천 탭: 개인화 행동 추천과 기대효과

설명 문구 예시:

검진AI 리셋코치는 건강검진 수치와 생활패턴을 입력하면 고혈압·당뇨·이상지질혈증 위험도를 산출하고, 사용자가 바로 실천할 수 있는 개선 행동을 제안합니다. 시제품은 웹앱 형태로 구현되어 있으며, 계정 기반 저장, 이전 분석 대비 변화 확인, OCR 입력 보조, 결측치 fallback, 공공데이터 샘플 입력 기능을 포함합니다.
