# 검진AI 리셋코치 제출용 서비스 요약

## 1. 서비스 한 줄 설명

검진AI 리셋코치는 건강검진 수치와 생활패턴을 바탕으로 고혈압·당뇨·이상지질혈증 위험도를 예측하고, 사용자가 오늘 바로 실천할 수 있는 작은 개선 행동을 제안하는 예방관리 웹앱입니다.

## 2. 해결하려는 문제

많은 사용자는 건강검진 결과지를 받아도 수치의 의미를 정확히 이해하기 어렵고, 생활습관을 어떻게 바꿔야 하는지 구체적인 행동으로 연결하지 못합니다. 특히 "운동하세요", "식단 조절하세요" 같은 일반적인 안내는 실천 난이도가 높고 지속성이 낮습니다.

검진AI 리셋코치는 검진 수치를 단순히 보여주는 데서 끝나지 않고, 위험 요인을 설명한 뒤 "점심 후 단 음료 1잔 줄이기", "식후 10분 걷기", "식사 전에 채소 조금 먼저 먹기"처럼 부담이 낮은 행동으로 연결합니다.

## 3. 핵심 사용자

- 건강검진 결과를 받았지만 수치 해석이 어려운 성인
- 당뇨, 고혈압, 이상지질혈증 위험을 미리 관리하고 싶은 사용자
- 극단적인 식단이나 고강도 운동이 아니라 생활 속 작은 개선부터 시작하고 싶은 사용자
- 건강 앱 연동 전에도 웹앱에서 간단히 검진 수치를 입력해 예방관리 피드백을 받고 싶은 사용자

## 4. 핵심 기능

- 건강검진 수치 입력
- 허리둘레 등 모르는 항목 미입력 처리
- 생활패턴 입력
- 익명 브라우저 ID 기반 이전 분석값 대비 변화 확인
- Firebase 기반 계정, 개인 기본정보, 이전 진료기록 저장
- 공공데이터포털 건강검진정보 샘플 불러오기
- 고혈압·당뇨·이상지질혈증 위험도 예측
- 위험도를 높인 주요 요인 설명
- 하루 1~2개의 실천 가능한 개선 행동 추천
- 1주 목표 체크리스트 제안
- 건강검진 결과지 OCR 업로드 데모 흐름

## 5. 데이터 활용

### 필수 데이터

- 국민건강보험공단_건강검진정보

활용 방식:

- 건강검진 항목 구조 파악
- 공복혈당, 혈압, BMI, 허리둘레, 지질 수치 등 위험예측 입력값 구성
- 공공데이터 샘플을 웹앱 입력값으로 변환하는 API 구현
- 향후 모델 학습용 데이터셋으로 사용

### 보조 데이터

- 국민건강보험공단_직역별 성별 연령별 건강검진정보

활용 방식:

- 성별·연령별 건강검진 통계 참고
- 사업계획서에서 시장성과 위험군 규모 설명에 활용 가능

## 6. AI 활용 방식

MVP에서는 검진 기준값과 생활패턴 가중치를 결합한 위험예측 엔진을 사용합니다. 동시에 국민건강보험공단 건강검진정보 CSV를 내려받아 RandomForest 또는 LogisticRegression 모델을 학습할 수 있는 파이프라인을 포함했습니다.

AI 활용 범위:

- 질환별 위험도 예측
- 위험 요인 설명
- 위험도가 높은 1~2개 요인을 우선 개선 대상으로 선택
- 사용자 생활패턴을 반영한 퍼스널 개선 플랜 생성
- 실천 난이도가 낮은 행동만 추천

## 7. 차별성

- 단순 건강정보 제공이 아니라 "위험예측 → 이유 설명 → 오늘 할 일"까지 한 화면에서 연결
- 극단적 식단이나 고강도 운동이 아닌 지속 가능한 행동 중심
- 공공데이터 기반으로 검진 항목과 예측 대상을 구성
- 웹앱 형태라 앱스토어·플레이스토어 승인 없이 빠르게 시연 가능
- 향후 OCR, HealthKit, Health Connect, 의료 마이데이터 연동으로 확장 가능

## 8. 안전 원칙

- 의료 진단 또는 치료 처방이 아님을 명확히 고지
- 약물 중단, 특정 치료 판단, 질병 확정 표현 금지
- 단기간 급격한 감량 목표 금지
- 하루 한 끼, 단백질 쉐이크만 섭취 같은 극단적 식단 금지
- 식후 바로 격한 러닝 같은 무리한 운동 금지
- 기저질환, 약물 복용, 알레르기가 있으면 의료진 상담 권고

## 9. 현재 구현 상태

- FastAPI 기반 웹앱 구현 완료
- Render 배포 완료
- GitHub 저장소 연결 완료
- 공공데이터포털 API 키 기반 건강검진정보 샘플 조회 확인
- 공공데이터 샘플을 입력폼으로 변환하는 API 구현
- 위험예측 및 퍼스널 플랜 생성 API 구현
- UXPilot 기반 UI 개선 및 분석 화면 적용

## 10. 향후 확장 계획

1. 건강검진 결과지 OCR 자동 입력
2. 실제 건강검진정보 CSV 기반 모델 학습 및 성능 지표 반영
3. HealthKit, Health Connect를 통한 걸음수·활동량·수면 연동
4. 건강정보고속도로 또는 의료 마이데이터 기반 검진 내역 연동
5. 사용자별 실천 기록 기반 추천 고도화
