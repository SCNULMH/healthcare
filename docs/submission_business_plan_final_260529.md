## 2. 창업아이템 개요

검진AI 리셋코치는 건강검진 결과를 받은 사용자가 자신의 만성질환 위험 신호를 이해하고 생활습관 개선을 시작하도록 돕는 웹 기반 예방관리 서비스이다. 많은 사용자는 건강검진표의 혈압, 공복혈당, 콜레스테롤 수치를 확인해도 어떤 수치가 중요한지, 지금 무엇을 바꾸면 좋은지 판단하기 어렵다. 또한 허리둘레나 지질 수치처럼 기억하지 못하는 항목이 있으면 기존 서비스에서는 분석 자체가 막히거나 부정확한 기본값으로 채워지는 문제가 있다.

본 서비스는 검진 입력, 생활 입력, 결과 계산, 이유 설명, 오늘의 행동, 저장/변화 확인을 하나의 흐름으로 제공한다. 사용자는 직접 검진 수치를 입력하거나 공공데이터 샘플을 불러올 수 있고, OCR을 통해 건강검진표 이미지/PDF에서 읽어낸 값을 입력 후보로 반영할 수 있다. 분석 결과는 고혈압·당뇨·이상지질혈증 위험 신호로 제공되며, 결과에 영향을 준 요인과 낮추기 위한 행동이 함께 표시된다.

핵심 고객은 건강검진을 받은 성인, 만성질환 전단계 관리가 필요한 사용자, 건강 앱 연동 전에도 웹에서 간편하게 검진 기반 예방관리를 시작하고 싶은 사용자이다. 서비스는 의료 진단이나 치료 결정을 대체하지 않고, 건강검진 기반 예방관리 참고 정보와 의료진 상담 권고를 제공한다.

## 3. 창업아이템의 독창성·발전가능성·완성도

### 독창성

검진AI 리셋코치의 차별점은 “위험 신호 확인 → 이유 설명 → 오늘의 행동”을 한 번에 연결한다는 점이다. 기존 서비스는 수치 기록, 운동 기록, 일반 건강정보 제공에 머무르는 경우가 많다. 본 서비스는 공복혈당, 혈압, 지질 수치, BMI, 생활패턴을 함께 보고 질환별 위험 신호를 계산한 뒤, 사용자가 바로 실행할 수 있는 작은 행동으로 바꿔준다.

또한 모르는 검진 항목을 다루는 방식이 다르다. 사용자가 수축기혈압만 알고 이완기혈압은 모르는 경우에도 입력된 혈압값은 판단에 반영한다. 지질 수치를 모두 모르는 경우에는 총콜레스테롤·HDL·LDL·중성지방 직접 판단을 제외하고, 외식·야식·운동·걸음수 등 생활요인을 기반으로 위험 신호를 계산한다. 사용자가 완전한 검진표를 가지고 있지 않아도 서비스 접근성을 유지한다.

### 발전가능성

단기적으로는 B2C 예방관리 웹앱으로 운영 가능하며, 중기적으로는 건강검진 수검자 대상 리포트 서비스, 기업 임직원 건강관리, 지자체 건강증진사업 연계로 확장할 수 있다. 장기적으로는 의료 마이데이터, 건강정보고속도로, HealthKit/Health Connect와 연동해 검진 기록·걸음수·활동량 자동 입력 기반의 개인 프로필 서비스를 제공할 수 있다.

수익 모델은 개인 프리미엄 리포트, 기업·기관 건강관리 SaaS, 보건소·지자체 예방관리 프로그램 연계, 보험·헬스케어 파트너십 등을 검토할 수 있다. 민감 건강정보를 다루므로 사용자 동의, 최소수집, 암호화, 비식별 통계 활용 원칙을 우선한다.

### 완성도

현재 시제품은 FastAPI 백엔드와 정적 웹 UI로 구현되어 있다. 주요 기능은 건강검진 수치 입력, 생활패턴 입력, 항목별 모름 처리, 위험 신호 분석, 결과 탭, 평균 비교, 입력 완성도 카드, 내 프로필 맞춤 행동, Firebase 계정·프로필·이전 진료기록 저장, 분석 결과 저장, OCR 입력 보조, Render 배포 설정이다. 저장 API는 로그인 세션 토큰이 없으면 서버에서 차단하도록 설계했다.

검증은 Python 유닛 테스트와 JavaScript 문법 검사를 통해 수행한다. 모르는 검진 항목 처리, 음주 입력 변환, Firebase 인증 정보가 없을 때의 대체 저장 흐름, OCR 결과의 모름 자동 연동, 저장 인증 흐름을 테스트 케이스로 포함했다.

## 4. 보건의료분야 공공데이터 및 AI기술을 활용한 사업추진 계획

### 단기 추진 목표

- 제출용 웹 시제품 안정화 및 Render 배포

- 검진표 자동입력과 사용자 확인 흐름 개선
- 사업계획서 및 시제품 증빙 캡처 정리

### 중기 추진 목표

- 공공데이터 CSV 기반 학습 데이터셋 확장
- 당뇨·고혈압·이상지질혈증 모델별 성능 지표 보완
- 사용자별 저장 기록 기반 변화 추이 분석
- 보건소, 기업 건강관리, 지자체 건강증진사업 PoC 검토

### 장기 추진 목표

- 건강정보고속도로·의료 마이데이터 기반 검진 내역 연동
- HealthKit/Health Connect 기반 걸음수·활동량·수면 자동 연동
- 실천 기록 기반 내 프로필 추천 개선
- 개인정보 보호형 통계 리포트와 기관용 대시보드 확장

### 공공데이터 활용 방안

국민건강보험공단 건강검진정보는 공복혈당, 혈압, BMI, 허리둘레, 콜레스테롤, 중성지방 등 위험 신호 계산에 필요한 입력값을 구성하는 기준 데이터로 활용한다. 국민건강보험공단 직역별 성별 연령별 건강검진정보는 연령대·성별 평균 비교 및 시장 규모 설명에 활용한다. 보건의료빅데이터개방시스템과 공공데이터포털의 보건의료분야 공개 데이터는 향후 질환 통계, 의료 이용 현황, 지역 건강관리 수요 분석으로 확장할 수 있다.

### AI 구현 방식

현재 분석 엔진은 설명 가능한 기준 기반 계산과 학습 모델 파이프라인을 병행한다. 검진 핵심값이 모두 명확하고 모델 파일이 준비된 경우에는 학습 모델을 적용할 수 있고, 값이 부족한 경우에는 검진 기준과 생활패턴 기준으로 계산한다. 결과 화면에는 입력값 완성도, 학습 평균과의 비교, 질환별 판단 근거를 표시해 사용자가 결과를 이해할 수 있도록 했다.

OCR 기능은 건강검진표 이미지/PDF에서 수치를 읽어 입력 후보로 반영한다. 원본 파일은 서버나 DB에 영구 저장하지 않으며, 읽지 못한 항목은 “모름”으로 자동 체크하여 사용자가 최종 확인한 뒤 분석하도록 설계했다.

## 5. 기대효과

### 기술적 파급효과

공공 건강검진 데이터를 단순 조회가 아니라 내 검진 결과를 이해하고 행동으로 옮기는 경험으로 전환한다. 모르는 항목 처리, 검진표 자동입력, 설명 가능한 분석, 기록 기반 변화 비교를 결합해 실제 사용자 환경에서도 지속 가능한 건강관리 시제품을 제시한다.

### 산업적 파급효과

검진기관, 기업 건강관리, 지자체 건강증진사업, 보험·헬스케어 플랫폼과 연계할 수 있는 확장성이 있다. 수검자는 검진 결과를 이해하고 행동으로 연결할 수 있고, 기관은 예방관리 프로그램 참여율과 사후관리 효율을 높일 수 있다.

### ESG 혁신

건강정보 접근성과 이해도를 높여 예방 중심의 사회적 가치를 창출한다. 웹 기반 서비스로 별도 앱 설치 없이 접근 가능하며, 종이 안내문 중심의 건강관리 정보를 디지털 리포트로 전환할 수 있다. 또한 과도한 의료 진단 표현을 피하고 사용자 동의·최소수집·비저장 OCR 원칙을 적용해 윤리적 AI 활용을 지향한다.
