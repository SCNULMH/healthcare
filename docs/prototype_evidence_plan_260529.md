# 시제품 증빙 자료 작성본

대회: 보건의료빅데이터·AI 활용 창업경진대회  
서비스명: 검진AI 리셋코치  
공모 분야: 제품 및 서비스 개발

## 필수 설명 문구

검진AI 리셋코치는 건강검진 수치와 생활패턴을 입력하면 고혈압·당뇨·이상지질혈증 위험도를 산출하고, 사용자가 바로 실천할 수 있는 개선 행동을 제안하는 웹 시제품이다. 시제품은 계정 기반 저장, 이전 분석 대비 변화 확인, OCR 입력 보조, 결측치 fallback, 공공데이터 샘플 입력 기능을 포함한다.

본 서비스는 의료 진단 또는 치료 처방을 제공하지 않는다. 결과는 건강검진 기반 예방관리 참고 정보이며, 위험 신호가 높거나 증상이 있는 경우 의료진 상담을 안내한다.

## 첨부 캡처 순서

| 순서 | 화면 | 증빙 포인트 |
| ---: | --- | --- |
| 1 | 홈 화면 | 서비스명, 건강 리셋 점수, 이전 분석 기록 영역 |
| 2 | 계정 화면 | 로그인/회원가입, 로그인 후 폼 숨김 상태 |
| 3 | 기본정보 입력 | 나이, 성별, 키, 체중, 허리둘레 입력 |
| 4 | 검진 입력 | 혈압·혈당·지질 수치와 항목별 모름 체크 |
| 5 | 활동/생활 입력 | 걸음수, 운동, 아침식사, 단 음료, 야식, 외식, 음주 입력 |
| 6 | OCR 화면 | 건강검진표 업로드, 원본 비저장 안내 |
| 7 | 결과 기준 탭 | AI 판단 단계, 생활요인 기준, 평균 비교 |
| 8 | 위험도 탭 | 고혈압·당뇨·이상지질혈증 위험도 카드 |
| 9 | 신뢰도 탭 | 입력값 완성도와 분석 신뢰도 |
| 10 | 추천 탭 | 개인화 행동 추천과 기대효과 |
| 11 | 저장/기록 | 저장 후 이전 분석 기록 또는 변화 요약 |
| 12 | API 상태 | `/health/ready`, `/health/public-data`, `/account/status` 중 1개 |

## 캡처별 짧은 설명

### 홈 화면

서비스 첫 화면에서 최근 건강 리셋 점수, 이전 분석 기록, 공공데이터 샘플 불러오기 진입점을 확인할 수 있다.

### 검진 입력

건강검진표에 포함된 혈압, 공복혈당, 지질 수치를 직접 입력한다. 사용자가 모르는 항목은 “모름”으로 표시해 해당 직접값만 분석에서 제외한다.

### 활동/생활 입력

최근 일주일 기준의 걸음수, 운동, 아침식사, 단 음료, 야식, 외식과 음주 세부값을 입력한다. 생활패턴은 위험도 보정과 개인화 추천 행동 생성에 사용된다.

### OCR 입력 보조

건강검진표 이미지/PDF를 업로드하면 OCR 입력 후보를 생성한다. 원본 파일은 영구 저장하지 않고, 사용자가 확인한 값만 분석에 반영한다.

### 분석 결과

질환별 위험도와 판단 근거, 입력값 완성도, 평균 비교, 추천 행동을 탭 형태로 확인할 수 있다. 결과는 의료 진단이 아니라 예방관리 참고 정보로 안내된다.

## 제출 전 보안 체크

- Firebase admin JSON, API key, `.env` 내용 캡처 금지
- 브라우저 주소창에 민감 토큰이 보이면 가림
- 개인 이름, 이메일, 전화번호가 화면에 보이면 가림
- 의료 진단·처방처럼 보이는 과장 문구가 없는지 확인
- PDF 저장 후 전체 파일을 10MB 이하 ZIP으로 압축
