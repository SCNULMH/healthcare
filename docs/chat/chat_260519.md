## 2026-05-19 Render UI 미반영 원인 확인 기록

### 원인
- UI 변경 작업 자체는 `static/index.html`, `static/styles.css`, `static/app.js`에 반영되어 있었고 GitHub `main`에도 push되어 있었다.
- 그러나 `render.yaml`에 `autoDeployTrigger: off`가 설정되어 있어 Render가 GitHub push를 자동 배포하지 않는 상태였다.
- 따라서 사용자가 Render 배포 URL에서 확인한 화면은 최신 커밋이 아니라 이전 배포본일 가능성이 높다.

### 조치
- `render.yaml`의 `autoDeployTrigger`를 `commit`으로 변경.
- 이후 커밋부터 linked branch push 시 Render가 자동 배포하도록 설정.

### 사용자 확인 필요
- Render Dashboard에서 현재 서비스의 Manual Deploy 또는 Deploy latest commit을 한 번 실행해야 기존 미반영 UI가 즉시 반영될 수 있다.
- 이후에는 `autoDeployTrigger: commit` 설정이 적용되면 push마다 자동 배포된다.
