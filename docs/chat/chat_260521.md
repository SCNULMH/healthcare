- 규칙 기반 엔진은 당뇨, 고혈압, 이상지질혈증 위험도를 모두 산출하고 개인 맞춤 개선 행동을 생성.
- `scripts/train_risk_models.py`는 공공데이터 CSV 기반 RandomForest/LogisticRegression 학습 파이프라인으로 구현되어 있음.

### 발견한 문제
- 기존 학습 스크립트는 지질 수치 결측이 있으면 `dropna()`로 전체 행을 제거해, 샘플 CSV에서 학습 데이터가 0건이 되는 문제가 있었음.
- 공공데이터 API에서 내려받은 건강검진정보 3,000건 표본은 총콜레스테롤, HDL, LDL, 중성지방 값이 모두 비어 있어 이상지질혈증 학습 라벨을 만들 수 없었음.
- 당뇨/고혈압 모델의 높은 성능은 공복혈당·혈압 기준으로 만든 라벨을 같은 검진 수치로 분류하기 때문이므로, 장기 질병 발생 예측 성능으로 설명하면 안 됨.

### 수정 내용
- `scripts/train_risk_models.py`
  - 결측 feature 전체 제거 대신 target 라벨 기준으로만 필터링하도록 수정.
  - 수치형 변수에 median imputer 적용.
  - 전부 결측인 수치 컬럼도 feature shape가 유지되도록 `keep_empty_features=True` 적용.
  - 학습 표본 부족, 단일 클래스, 소수 클래스 부족 시 실패하지 않고 `skipped` 리포트로 기록하도록 수정.
  - 이상지질혈증은 지질 수치가 있는 행에서만 라벨을 생성하도록 수정.
- `docs/ai_model_strategy.md`
  - 현재 라벨은 검진 수치 기준 위험군 분류이며 미래 질병 발생 예측이 아니라는 주의 문구 추가.
  - 지질 수치 결측 시 이상지질혈증 모델은 학습 제외되고 MVP 규칙 기반 산출을 유지한다는 설명 추가.
