# 제출서류 작성 초안

## 서비스명

검진AI 리셋코치

## 한 줄 소개

건강검진 기반 위험예측 및 실천 가능한 퍼스널 개선 플랜 웹앱

## 서비스 개요

검진AI 리셋코치는 사용자의 건강검진 수치와 생활패턴을 입력받아 고혈압, 당뇨, 이상지질혈증 위험도를 예측하고, 실생활에서 바로 실천할 수 있는 하루 1~2개의 작은 개선 행동을 제안하는 예방관리 보조 서비스입니다.

본 서비스는 의료 진단이 아닌 예방관리 보조 서비스입니다. 질병 확정, 약물 조정, 치료 처방을 대신하지 않으며, 위험 신호가 높거나 기저질환·약물 복용 이력이 있는 사용자는 의료진 상담을 안내합니다.

## 활용 공공데이터

- 국민건강보험공단_건강검진정보
- 국민건강보험공단_직역별 성별 연령별 건강검진정보

활용 방식:

- 공복혈당, 혈압, BMI, 허리둘레, 콜레스테롤, 중성지방 등 건강검진 항목 구조를 서비스 입력값으로 구성
- 공공데이터 샘플을 웹앱 입력값으로 변환해 시연 데이터로 사용
- 향후 CSV 기반 모델 학습 데이터로 활용
- 성별·연령별 건강검진 통계를 위험군 규모와 시장성 설명에 활용

## AI 활용 내용

MVP는 설명 가능한 규칙 기반 위험예측 엔진으로 구현했습니다. 공복혈당, 혈압, 지질 수치, BMI, 허리둘레, 걸음수, 운동 빈도, 단 음료 섭취 등 주요 요인을 질환별로 점수화해 위험도를 산출합니다.

확장 구조도 함께 구현했습니다. `scripts/train_risk_models.py`를 통해 국민건강보험공단 건강검진정보 CSV 기반 RandomForest 또는 LogisticRegression 모델을 학습할 수 있으며, API와 UI는 향후 학습 모델로 교체해도 유지되도록 분리했습니다.

AI 활용 범위:

- 고혈압·당뇨·이상지질혈증 위험도 예측
- 위험도를 높인 주요 요인 설명
- 우선 개선할 위험 요인 1~2개 선택
- 사용자의 생활패턴과 제약을 반영한 퍼스널 개선 플랜 생성
- 실천 난이도와 기대효과를 쉬운 문장으로 설명

## 차별성

검진AI 리셋코치는 단순히 건강검진 수치를 보여주는 서비스가 아니라, 위험도 예측 결과를 사용자가 오늘 실행할 수 있는 행동으로 바꿉니다.

추천 방식은 극단적 식단·운동이 아닌 하루 1~2개의 작은 행동 중심입니다. 예를 들어 혈당 위험이 높은 사용자는 점심 후 단 음료 1잔 줄이기, 식사 전에 채소 조금 먼저 먹기, 저녁 식후 10분 걷기처럼 부담이 낮은 행동부터 시작합니다.

## 구현 범위

- 웹앱 UI
- 수동 입력 기반 건강검진 수치 입력
- 공공데이터 샘플 불러오기
- 고혈압·당뇨·이상지질혈증 위험도 분석
- 위험 요인 설명
- 하루 개선 행동 추천
- 1주 체크리스트 생성
- OCR 업로드 데모 화면과 API 확장 지점
- Render 배포 구조

## OCR 및 연동 로드맵

이번 제출에서는 OCR을 핵심 완성 기능이 아니라 다음 단계 확장 기능으로 둡니다.

MVP는 수동 입력과 공공데이터 샘플 입력을 우선 구현했습니다. OCR은 입력 부담을 줄이기 위한 다음 단계 기능이며, 실제 적용 시 OCR 결과를 바로 예측에 쓰지 않고 사용자가 확인·수정한 뒤 반영합니다.

실제 OCR 구현 후보는 Naver CLOVA OCR 또는 OpenAI Vision을 우선 검토합니다. 업로드 파일은 영구 저장하지 않고, 외부 OCR API 사용 시 사용자 동의를 받은 뒤 필요한 수치만 분석에 사용합니다.

향후 HealthKit, Android Health Connect, 의료 마이데이터 연동을 통해 걸음수와 건강검진 내역 자동 입력을 확장할 수 있습니다.

## 안전 원칙

- 의료 진단 또는 치료 처방으로 표현하지 않음
- 약물 중단, 치료 변경, 질병 확정 표현 금지
- 극단적 식단·운동 추천 금지
- 단기간 급격한 감량 목표 금지
- 알레르기, 기저질환, 약물 복용 정보가 불명확한 경우 일반 예시로 안내
- 위험도가 높거나 증상이 있으면 의료진 상담 권고
