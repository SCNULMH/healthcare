## 사업 내용

건강검진 공공데이터 기반으로 사용자의 검진 수치와 생활패턴을 분석해 고혈압·당뇨·이상지질혈증 위험도를 예측하고, 사용자가 오늘 바로 실천할 수 있는 개인 맞춤 생활개선 행동을 제안하는 예방관리 웹 서비스입니다. 수동 입력, 공공데이터 샘플 입력, OCR 입력 보조, 계정 기반 기록 저장, 이전 분석 대비 변화 확인 기능을 시제품으로 구현했습니다.

## 공공데이터 활용 목록

- 국민건강보험공단_건강검진정보
- 국민건강보험공단_직역별 성별 연령별 건강검진정보
- 보건의료빅데이터개방시스템 및 공공데이터포털의 보건의료분야 공개 데이터

## 기대 효과

검진 결과를 이해하기 어려운 사용자가 주요 만성질환 위험요인을 쉽게 파악하고, 단순 정보 확인을 넘어 실천 가능한 생활개선 행동으로 연결할 수 있습니다. 의료기관 방문 전 예방관리 관심을 높이고, 공공 건강검진 데이터의 국민 체감형 활용 사례를 확장할 수 있습니다.
